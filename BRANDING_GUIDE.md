# AngiSoft Technologies - Branding Guide

## Logo
- **File**: `/frontend/public/logo.svg`
- **Usage**: Displayed in header, footer, and admin areas
- **Format**: SVG (scalable)
- **Colors**: Teal background with peach/white circles

## Brand Colors

### Primary Colors
- **Teal Primary**: `#14B8A6` (rgb(20, 184, 166))
- **Teal Dark**: `#0D9488` (rgb(13, 148, 136))
- **Teal Light**: `#5EEAD4` (rgb(94, 234, 212))

### Accent Colors
- **Peach**: `#FFB6A3` (from logo)
- **White**: `#FFFFFF`

### Usage Guidelines
- **Primary Actions**: Use teal (#14B8A6)
- **Hover States**: Use darker teal (#0D9488)
- **Text Gradients**: Teal to teal (400-600)
- **Links**: Teal with hover underline
- **Buttons**: Teal background, white text

## Typography
- **Font Family**: Poppins, system fonts
- **Headings**: Bold, gradient text (teal)
- **Body**: Regular weight

## Design Elements
- **Rounded Corners**: 8px (lg) for cards, 4px (md) for buttons
- **Shadows**: Subtle shadows on cards
- **Hover Effects**: Scale (1.05) and shadow increase
- **Transitions**: 0.3s ease for all interactions

## Implementation
All brand colors are defined in:
- `frontend/tailwind.config.js` - Tailwind color definitions
- `frontend/src/utils/constants.js` - Brand color constants
- Used consistently throughout all components
